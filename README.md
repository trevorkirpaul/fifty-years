# Fifty Years

## About

This is a game heavily inspired by [Fifty Years](https://echa.itch.io/) created by Echa. I found that game very fun and simple and decided to make my own version of it using React and TypeScript.

## Tech

- React
- Styled-Components
- TypeScript
